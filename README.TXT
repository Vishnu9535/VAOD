Voice Assisted Object_Detection for Visully impared(VAOD)

Its an voice  object_detection model for visually impared which gives
direction as well as distance of the objects detected from the camera input
VAOD files contains different files related to the model
audio.py gives the audio output 
derection.py gives the direction of the object from the camera
model.py is the implimentation of the final model
Run VAOD.py to run the final model
